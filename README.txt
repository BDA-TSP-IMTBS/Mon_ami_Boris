Ceci est un bot Discord ayant pour but de donner des rôles aux personnes réagissant à cetains messages.

Il faut rajouter à ce GitHub un fichier "config.json" de la forme:
{
  "token": "your-token"
}

Le token du bot étant comme son mot de passe, il ne peut pas être mis sur le GitHub

Les rôles à ajouter sont contrôlés par le fichier configReactionToRole.json



Pour lancer le bot, il faut lancer index.js comme ceci:

node ./index.js
